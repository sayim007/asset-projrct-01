# MyShop E-commerce Template üõí

This repository contains a modern, fully responsive e-commerce landing page template built primarily with **HTML5** and **Bootstrap 5**. It showcases a standard layout for an online store, including a sticky navigation bar, hero carousel, product categories, featured listings, and a subscription footer.

## üåü Features

* **Fully Responsive:** Optimized for desktop, tablet, and mobile views using the Bootstrap 5 grid system.
* **Sticky Navbar:** Navigation bar remains fixed at the top for easy access.
* **Hero Carousel:** Image slider to highlight promotions and new arrivals.
* **Product Grid:** Dedicated sections for categories, featured products, and new arrivals.
* **Call-to-Action (CTA):** Prominent sections for sales and newsletter subscription.
* **Font Awesome Integration:** Used for feature icons (Why Choose Us section).
* **Google Fonts:** Utilizes the Poppins typeface for a modern look.
* **Semantic HTML:** Clean, well-structured markup following accessibility best practices.

## üé® Customization

### 1. Styling

Custom styles can be modified in the `assets/all.css` file.

* To change the primary color scheme, you can override Bootstrap's CSS variables within `all.css` or change the specific utility classes (`bg-primary`, `btn-primary`, etc.) directly in `index.html`.

### 2. Images

Replace the placeholder images (`https://via.placeholder.com/`) and the demo images (`assets/media/images/...`) with your actual product and banner images.

### 3. JavaScript Components

Bootstrap's JavaScript is linked via the CDN at the bottom of the `<body>` tag in `index.html`:

```html
<script src="[https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js](https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js)"></script>
````

Ensure this link remains intact for the Navbar toggle, Carousel sliding, and Dropdowns to function correctly.
